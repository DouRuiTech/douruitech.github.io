后端开发工程师职位要求

Node 后端：

  • 两年及以上 Node 开发经验。
  • 扎实的后端知识（深入了解 Node.js、Linux 系统知识、熟悉常用数据库等）。
  • 丰富的项目经验 加分，有中大规模的 Node.js 线上产品研发经验，加分。
  • 熟悉大数据体系、有数据分析类产品使用/研发经验，加分。
  • 有开源项目，加分。

Android 职位要求

  • Android 工作经验一年以上，有独立开发 Android App 经验者优先。
  • 有良好的编程习惯，代码结构清晰，命名规范。
  • 熟练掌握 Java，熟悉 Android SDK，熟悉 Android 的 UI、网络和常用开发库。
  • 熟悉掌握 OpenGL ES 相关 API，了解视频录制、滤镜、后期剪辑、特效、视频编解码等相关开发。
  • 熟练使用 Git 进行协同开发，了解 Git Flow，具备良好的沟通能力和优秀的团队协作能力。
